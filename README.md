# SHS_app_ios Prototype

`prototype_color fix.html` 기반의 모바일 앱 스타일 인터랙티브 프로토타입입니다.

## 프로젝트 소개

이 프로젝트는 육아 커뮤니티/정보/스탠다드/마켓/프로필 흐름을 한 화면에서 시뮬레이션할 수 있도록 제작된 단일 HTML 프로토타입입니다.

- 모바일 앱 UI 형태의 단일 페이지 구성
- 탭 전환, 서브 화면 슬라이드, 모달, 댓글, 반응(리액션) 등 데모 인터랙션 포함
- `스탠다드` 탭은 성분 안전성 중심 정보 구조를 반영해 개선됨

## 실행 방법

### 1) 파일 직접 열기

브라우저에서 아래 파일을 바로 열어 확인할 수 있습니다.

- `prototype_color fix.html`

### 2) 로컬 서버 실행 (권장)

```bash
cd "/Users/elbert.kim/Desktop/aguard"
python3 -m http.server 8000
```

브라우저 접속:

- `http://localhost:8000/prototype_color%20fix.html`

## 주요 화면

- 커뮤니티: 전체/인기/주제별/월령별
- 육아정보: 카테고리 탭 기반 정보 카드
- 스탠다드: 기준 투표/인정템, 제품 상세(성분 안전점수/성분요약/대체제품 비교)
- 별똥별마켓: 리뷰약속/응모/교환/웰컴존
- 프로필: 등급/미션/설정
